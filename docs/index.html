How to install the APP

Start data API

1/ Take the Datas API from https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard.git

2/ Follow api documentation to start it

Start SportSee app

1/ Take the SportSee project from https://github.com/SVergnaudExpleo/stephanevergnaud_9_20211018.git

2/ Open SportSee project and install all dependencies with NPM Install.

3/ Configure the user ID in ApiCall/index Choose between 12 and 18.

4/ Start project with NPM Start or YARN Start

5/ enjoy our app
